# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, and keep discussion focused on the specification and its technical merits. Harassment and discrimination are not tolerated.

## Enforcement

Report unacceptable behavior to `conduct@dekimu.com`. Reports are handled confidentially. The Steward may remove, edit, or reject contributions and may ban contributors who violate this code.

The full text of the Contributor Covenant 2.1 governs; this file is a pointer to it.
